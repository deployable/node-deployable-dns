# [deployable-dns](https://github.com/deployable/deployable-dns)

Custom DNS server

### Install
 
    npm install deployable-dns --save

    yarn add deployable-dns

### Usage

```javascript

const deployable-dns = require('deployable-dns')
deployable-dns.demo()

```
